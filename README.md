# Expressões regulares
Material com o qual foram estudadas as Expressões Regulares.

## Aplicação
* Entender o que são expressões regulares;
* Validar formulários HTML;
* Identificar E-mail, CPF, CEP.

## Conceitos
* Regex Engine
* Meta Caracteres
* Classes de Caracteres
* Quantifiers
* Âncoras
* Grupos
* Backreference
